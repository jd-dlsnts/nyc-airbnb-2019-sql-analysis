# NYC Airbnb Market Analysis

## Overview
SQL-based analysis of 48,000+ Airbnb listings across New York City 
using the AB_NYC_2019 dataset from Kaggle.

## Business Question
How does pricing, demand, and host concentration vary across 
NYC boroughs and neighbourhoods?

## Tools Used
- SQLite / DB Browser for SQLite
- Google Docs

## Key Findings
- Brooklyn leads NYC in review activity at 42.8% of all reviews
- Midtown ranks only 10th in listings despite being the primary tourist corridor
- Corporate operators dominate the top host tier
- Cobble Hill and Brooklyn Heights command prices competitive with Manhattan averages

## Files
- queries.sql — all 10 SQL queries with comments
- NYC_Airbnb_Analysis.pdf — full findings document with executive summary and recommendations

## Dataset
[NYC Airbnb Open Data](https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data)
